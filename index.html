This is the link section

There are three types of it

  • The first one is Unordered Link, marked by the "ul" tag
  • The next one is Ordered Link, marked by the "ol" tag
  • The last one is Description Link, marked by the "dl" tag

UL and OL share the same tag "li" to list its item, whereas DL uses "dt" for the title and "dd" for the desc of each item in the list

This is the meta tag section

This section will introduce you another tag in HTML: the "meta" tag.
It's used to define some data about data for the web. It's always go inside the "head" element

There are some attribute of it

  • charset: Specifies the character encoding for the HTML document
  • content: Uses to define the value for these following attributes
    • http-equiv: Provides header for info/value of the content attribute
    • name: Specifies a name for the metadata

You can checkout this page source code to know how it's used
